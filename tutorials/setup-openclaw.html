Setting up OpenClaw: The Ultimate Personal Agent

A comprehensive guide to installing, configuring, and deploying your own autonomous OpenClaw agent locally or on a VPS.

OpenClaw is more than just a chatbot—it's an operating system for agentic AI. By the end of this tutorial, you'll have a fully functional agent running on your machine.

Prerequisites

  • Node.js v20+
  • A terminal (PowerShell, Bash, or Zsh)
  • API Keys (OpenAI, Anthropic, or Google)

Step 1: Installation

First, we need to install the OpenClaw core globally on your machine. Open your terminal and run:

npm install -g openclaw

Verify the installation by checking the version:

openclaw --version

Step 2: Initialization

Create a new workspace for your agent. This is where its memory and configuration will live.

mkdir my-agent
cd my-agent
openclaw init

You'll be prompted to select a template. Choose 'Personal Assistant' for a general-purpose agent.

Step 3: Configuration

Open the generated config.toml file. You need to add your LLM provider keys here.

[llm]
provider = "anthropic"
model = "claude-3-5-sonnet-20240620"
api_key = "sk-ant-..."

Step 4: Running the Agent

Launch your agent in interactive mode:

openclaw run

You should now see the OpenClaw banner and a prompt waiting for your command!

Pro Tip: Connecting to Telegram

To make your agent accessible from your phone, run the gateway service:

openclaw gateway connect --channel telegram --token YOUR_BOT_TOKEN
