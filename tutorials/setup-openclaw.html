Setting up OpenClaw

OpenClaw is your personal AI operating system. Follow these steps to get started:

1. Installation

npm install -g openclaw

2. Configuration

Run the init command to set up your workspace:

openclaw init

3. Connecting Models

OpenClaw supports Gemini, Claude, and local models. Edit your config.json to add your API keys.
