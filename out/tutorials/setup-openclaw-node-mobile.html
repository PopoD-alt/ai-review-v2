Setting up your first OpenClaw node on Android/iOS

Turn your mobile device into a powerful AI node. This allows OpenClaw to use your phone's camera, screen, and local compute.

Prerequisites
  • OpenClaw Desktop installed and running
  • Android (10+) or iOS (15+) device
  • Both devices on the same local network
1

1. Download & Install

Download the OpenClaw Mobile APK from the official GitHub releases (for Android) or join the TestFlight beta (for iOS). Install the app and grant the requested permissions for Camera and Overlay.

Alfred's Pro-Tip

The "Display over other apps" permission is required for the Canvas surface to appear over your mobile UI.

2

2. Enable Gateway Pairing

On your Desktop terminal, run the pairing command to generate a secure connection token and QR code.

bash
openclaw nodes pairing --show-qr
Alfred's Pro-Tip

If your firewall is active, ensure port 4444 is open for local peer-to-peer communication.

3

3. Scan and Connect

Open the OpenClaw app on your phone, tap "Pair Node", and scan the QR code on your monitor. The node should appear in your Desktop "Nodes" list immediately.

Alfred's Pro-Tip

Once paired, you can give your node a nickname like "Pixel-Node" or "iPhone-Primary" to easily target it in scripts.

4

4. Enable Canvas Surface

In the app settings, toggle "Enable Canvas". This allows OpenClaw to draw a transparent UI layer over your phone, which it uses to show its "thoughts" or highlight UI elements it is interacting with.

bash
openclaw nodes notify --target "my-phone" --title "Hello" --body "Canvas is active!"
